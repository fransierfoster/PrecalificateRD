# PrecalificateRD

Simulador hipotecario de Perfect House SRL.

- `index.html` — sitio completo (HTML + CSS + JS, single-file).
- Desplegado en Vercel: cada push a `main` actualiza automáticamente https://precalificaterd-v11.vercel.app y los dominios precalificaterd.com / .net / .do.

## Cómo actualizar el sitio

1. Edita `index.html`.
2. `git add index.html && git commit -m "Descripción del cambio" && git push`
3. Vercel despliega automáticamente en 1-2 minutos.
